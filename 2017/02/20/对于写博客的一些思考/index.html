对于写博客的一些思考

发表于 2017-02-19 | |

一开始跟着同学使用博客园，后面自己用了csdn，之前写的东西确实太渣。可能是还没有意识到写作博客的意义。

看了刘未鹏的<<暗时间>>,越来越觉得有必要好好想一想写作blog的意义。
就自己来说，之前的blog都比较水。
存在以下几个问题

  • 内容不完整

    只是把自己的过程写出来，对于这样做可以起到什么效果，某些步骤的关键点在哪里没有具体写
开篇没有一个提纲，诸如写清楚需要准备的东西之类。

  • 内容表达不简要

    当然这点需要多写，多读。segmentfault 里面有好多教程写的就比较完备
也就是在具备可操作性的同时，表达言简意赅。

  • 内容比较随意

    往往是自己有什么想法，没有深思熟虑就写出来，比如hexo的相关问题，其实可以分成几个部分来写。后期完善。

坚持写一个blog,更多的是为了以下几个方面

  • 用博客来记录自己的思考
    时间长了，可以作为纵向比较来评判自己的认识水平
  • 教比学好
    写教程的时候会尽量把一件事情写清楚。可能自己会操作了，但是有些模糊的步骤时间长了，自己也会忘。也就是把模糊意识里面的东西变成显意识。
  • 用来结实志同道合的博友
    这点我还需要积累，毕竟博客的质量还需要提高
